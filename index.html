March 18, 2026 — 3 Days Ago
x402
HTTP Status Code 402: Payment Required
Reserved since 1991. Unused for 35 years. Tempo Labs + Coinbase just gave it full semantics.
🤖
AI Agents Pay
Autonomous payment for any HTTP-accessible resource. No human checkpoint.
💱
Method Agnostic
Stablecoins, Lightning, card charges — any payment network registers a method ID.
🆓
Open Standard
Royalty-free. Being built into Coinbase SDKs now. Define the interface, not the implementation.
How the Protocol Works
1
Client sends standard HTTP GET request
↓
2
Server responds: HTTP 402 + payment challenge (amount, currency, recipient)
↓
3
Client fulfills payment — method agnostic: stablecoin, Lightning, card charge
↓
4
Client resubmits with cryptographic proof of payment
↓
5
Server verifies and responds 200 OK + resource + signed receipt
No bank. No payment processor. No human. Every API call, data fetch, and AI inference can now natively carry payment.
Technologies Compete. Protocols Converge.
In a protocol market, winner-take-all is not a possibility — it is the inevitable end state.
Network value grows with the square of participants (Metcalfe's Law).
📡
TCP/IP vs OSI
OSI was more elegant, had more backing. TCP/IP had more deployment. Adoption beats quality — first to critical mass wins.
📼
VHS vs Betamax
Betamax was technically superior. VHS achieved rental library dominance first. Inferior protocol, first to network mass, wins.
🌐
HTTP vs Gopher
1993: Gopher traffic exceeded HTTP. Minnesota announced licensing fees. CERN: HTTP royalty-free forever. Gopher dead by 1996.
Charging for protocol access taxes your own network effects. Openness accelerates adoption.
